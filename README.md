## Getting started

### Dependencies

* Python 3.10.x

### Installing

* Clone the repo locally
  
  ```bash
  git clone https://github.com/alittlebroken/expense-tracker.git
  ```


### Executing

* Perform the following
  
  ```bash
    cd expense-tracker
    python3 expenses.py
  ```
